## Funcionalidades

- Calculo automatico do total (`valor_hora x horas_estimadas`)
- Validacao de prazo (`data_fim` nao pode ser menor que `data_inicio`)
- Campo de valor/hora com:
  - Slider
  - Digitacao manual sincronizada
- Campo de descricao do servico com contador de caracteres
- Resumo detalhado do orcamento
- Exportacao em PDF (via janela de impressao)
- Layout responsivo com boa experiencia em desktop e mobile

## Como Executar

1. Coloque a pasta em:
   - `c:\xampp\htdocs\Sistema_Orcamento_V1`
2. Inicie o Apache no XAMPP.
3. Acesse:
   - `http://localhost/Sistema_Orcamento_V1/`
